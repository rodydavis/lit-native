# Lit Native

Run your Lit PWA everywhere.

- 2 way communication
- no node js or JavaScript runtime
- native web views
- native code via events
- resize / dark mode support
- no dependencies

## Supported Platforms

- iOS
- MacOS

## Getting Started

1. `npm run install`
2. `npm run build`
3. Open target folder (i.e. darwin, ios)
4. Run project
